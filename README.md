## Multi-platform image

Following platforms for this image are available:

```
$ docker run --rm mplatform/mquery crazymax/alpine-s6:latest
Image: crazymax/alpine-s6:latest
 * Manifest List: Yes
 * Supported platforms:
   - linux/amd64
   - linux/arm/v6
   - linux/arm/v7
   - linux/arm64
   - linux/386
   - linux/ppc64le
   - linux/s390x
```
